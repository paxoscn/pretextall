🎨 样式保留测试页面

测试说明

目的：验证文字避开图片时，所有原有样式（颜色、背景、字体、边框等）都被保留。

操作：移动鼠标到下面各个彩色段落上，观察文字重排时样式是否保持不变。

1. 紫色文字

这段文字是紫色的。当你移动鼠标到这里时，文字会重新排列以避开图片， 但是紫色应该保持不变。pretext 只改变文字的布局，不会影响任何 CSS 样式。 试着在不同位置移动鼠标，看看紫色是否一直保持。这个测试很重要， 因为它验证了我们的实现不会破坏原有的视觉设计。

2. 渐变背景文字

这段文字有漂亮的渐变背景。当文字重新排列时，背景渐变应该保持不变。 移动鼠标到这里，观察背景色是否依然是从绿色到蓝色的渐变。 这个测试验证了背景样式的保留。文字的布局改变了，但是背景依然美丽。 这就是 pretext 的强大之处 - 它只关注文字的位置，不会干扰其他样式。

3. 斜体衬线字体

这段文字使用 Georgia 衬线字体，并且是斜体的粉红色。 当鼠标移动到这里时，字体、颜色、斜体样式都应该保持不变。 只有文字的排列方式会改变，以避开图片。这展示了 pretext 如何与各种字体样式完美配合。无论是衬线字体还是无衬线字体， 无论是正体还是斜体，都能正确处理。

4. 带边框和背景的文字

这段文字有绿色边框和浅绿色背景。当文字重新排列时， 边框和背景色都应该保持不变。移动鼠标测试一下。 这个测试验证了盒模型相关的样式（border、padding、background） 在文字重排时不会受到影响。这对于保持页面的整体设计非常重要。

5. 渐变文字（需要支持的浏览器）

这段文字本身就是渐变色的（使用 background-clip）。 当文字避开图片时，渐变效果应该保持。这是一个高级的 CSS 技巧， 测试我们的实现是否能处理这种复杂的样式。移动鼠标看看效果。

6. 带阴影的文字

这段橙色文字有阴影效果。当文字重新排列时，阴影应该跟随文字一起移动。 text-shadow 是一个常用的样式，我们需要确保它在文字重排后依然有效。 移动鼠标到这里，观察阴影是否保持在文字下方。这个测试验证了 文字装饰效果的保留。

7. 等宽字体（代码风格）

这段文字使用等宽字体，模拟代码编辑器的样式。 背景是深色的，文字是绿色的，就像终端一样。 当文字避开图片时，这种代码风格应该完全保留。 这对于技术文档和代码展示页面很重要。 移动鼠标测试一下，看看是否保持了黑客风格。

8. 普通段落（对照组）

这是一个普通的段落，没有特殊样式。它使用默认的黑色文字和白色背景。 这个段落作为对照组，让你可以比较有样式和无样式的文字在避开图片时的表现。 移动鼠标到这里，观察文字如何重新排列。然后对比上面那些有样式的段落， 看看它们的样式是否都被正确保留了。

✅ 检查清单

移动鼠标到每个段落上，确认：

  • ✓ 文字颜色保持不变
  • ✓ 背景颜色/渐变保持不变
  • ✓ 字体样式（字体族、大小、粗细、斜体）保持不变
  • ✓ 边框和内边距保持不变
  • ✓ 文字阴影保持不变
  • ✓ 其他所有 CSS 样式保持不变
  • ✓ 只有文字的排列方式改变了

调试信息

打开浏览器控制台（F12）查看：
- 插件是否正确初始化
- 是否扫描到了所有段落
- 是否有任何错误信息

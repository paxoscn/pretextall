## 💡 提示

### 文件命名规则
- `.md` - Markdown 格式文档
- `.html` - 网页文件（浏览器打开）
- `.txt` - 纯文本文件
- `.js` - JavaScript 代码
- `.css` - 样式文件
- `.json` - 配置文件

### 阅读建议
- 新手：从 `.txt` 和简短的 `.md` 开始
- 进阶：阅读完整的 `.md` 文档
- 开发：查看代码文件和开发文档

### 打开方式
- `.md` 文件：文本编辑器或 Markdown 阅读器
- `.html` 文件：浏览器
- `.txt` 文件：任何文本编辑器
- 代码文件：代码编辑器（如 VS Code）

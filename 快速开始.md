# 🚀 快速开始

## 一、生成图标（2 分钟）

1. 双击打开 `generate-icons.html` 文件
2. 点击每个图标下方的"下载"按钮
3. 将下载的 3 个文件放入 `icons` 文件夹

## 二、安装插件（3 分钟）

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择这个文件夹
6. 完成！

## 三、开始使用（1 分钟）

1. 点击浏览器工具栏的插件图标
2. 确保"启用动画"开关已打开
3. 访问任意网页（如百度、知乎）
4. 移动鼠标，看动画被文字推开！

## 四、自定义（可选）

### 上传自己的图片
- 点击"选择图片"按钮
- 选择一张透明背景的 PNG 图片
- 推荐尺寸：100x100 到 500x500 像素

### 调整参数
- **动画大小**：控制动画元素的尺寸
- **互动强度**：控制文字推开动画的力度
- **影响半径**：控制多远的文字会影响动画

## 五、推荐设置

### 日常浏览
```
大小：50px
强度：40
半径：100px
```

### 演示展示
```
大小：80px
强度：70
半径：150px
```

### 专注阅读
```
大小：40px
强度：30
半径：80px
```

## ❓ 遇到问题？

### 图标不显示
→ 确保 icons 文件夹中有 3 个图标文件

### 动画不出现
→ 检查"启用动画"开关，刷新网页（按 F5）

### 效果不明显
→ 增加"互动强度"和"影响半径"

### 页面卡顿
→ 减小"影响半径"，降低"互动强度"

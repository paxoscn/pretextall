# 安装指南

## 📋 准备工作

在安装插件之前，你需要：

1. Chrome 浏览器（版本 88 或更高）
2. 插件的所有文件
3. 插件图标文件（可选，但推荐）

## 🎨 第一步：生成图标（可选）

如果 `icons` 文件夹中还没有图标文件：

1. 在浏览器中打开 `generate-icons.html`
2. 点击每个图标下方的"下载"按钮
3. 将下载的文件保存到 `icons` 文件夹中

或者使用你自己的图标文件，确保命名为：
- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)
- `icon128.png` (128x128 像素)

## 🔧 第二步：安装插件

### 方法 A：开发者模式（推荐用于测试）

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在页面右上角开启"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择包含 `manifest.json` 的项目文件夹
6. 点击"选择文件夹"

✅ 安装成功！你会在浏览器工具栏看到插件图标。

### 方法 B：打包安装（推荐用于分发）

1. 在 `chrome://extensions/` 页面
2. 确保"开发者模式"已开启
3. 点击"打包扩展程序"
4. 在"扩展程序根目录"中选择项目文件夹
5. 点击"打包扩展程序"按钮
6. 会生成两个文件：
   - `.crx` 文件（插件包）
   - `.pem` 文件（私钥，请妥善保管）
7. 将 `.crx` 文件拖入 Chrome 浏览器窗口
8. 点击"添加扩展程序"

## 🎮 第三步：使用插件

1. 点击浏览器工具栏的插件图标
2. 在弹出的设置面板中：
   - 确保"启用动画"开关已打开
   - （可选）点击"选择图片"上传自定义动画图片
   - 调整大小、强度、半径等参数
3. 访问任意网页，移动鼠标查看效果！

## 🔍 验证安装

安装成功后，你应该能看到：

- ✅ 浏览器工具栏出现插件图标
- ✅ 点击图标能打开设置面板
- ✅ 在网页上移动鼠标时能看到动画效果
- ✅ 动画会被文字"推开"

## ❌ 常见问题

### 问题 1：插件图标不显示

**解决方案：**
- 检查 `icons` 文件夹中是否有图标文件
- 确保图标文件名正确（icon16.png, icon48.png, icon128.png）
- 尝试重新加载插件（在扩展管理页面点击刷新按钮）

### 问题 2：动画不显示

**解决方案：**
- 确保在设置面板中"启用动画"开关已打开
- 刷新网页（按 F5）
- 检查浏览器控制台是否有错误信息（按 F12）

### 问题 3：无法加载插件

**解决方案：**
- 确保所有文件都在同一个文件夹中
- 检查 `manifest.json` 文件是否存在且格式正确
- 确保 Chrome 版本足够新（建议 88+）

### 问题 4：某些网站上不工作

**解决方案：**
- 某些网站（如 Chrome 网上应用店）出于安全原因不允许插件运行
- 尝试在其他普通网站上测试
- 检查网站是否使用了 Shadow DOM（可能影响文字检测）

## 🔄 更新插件

如果你修改了插件代码：

1. 在 `chrome://extensions/` 页面
2. 找到"文字互动动画"插件
3. 点击刷新按钮（🔄）
4. 刷新正在使用插件的网页

## 🗑️ 卸载插件

1. 在 `chrome://extensions/` 页面
2. 找到"文字互动动画"插件
3. 点击"移除"按钮
4. 确认删除

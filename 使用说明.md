# 文字互动动画 Chrome 插件

基于 [pretext](https://github.com/chenglou/pretext) 库实现的 Chrome 浏览器插件，实现**文字环绕图片**的动态布局效果。

## ✨ 效果说明

当你移动鼠标到网页文字上时，文字会**实时重新排列**以环绕鼠标光标（图片）。

### 核心特性

- ✅ **真正的文字环绕**：文字会在图片的左右两侧同时显示
- ✅ **智能拆分**：每一行文字会被拆分成左右两部分，环绕图片
- ✅ **保留所有样式**：颜色、字体、背景、边框等所有 CSS 样式都保持不变
- ✅ **实时响应**：鼠标移动时文字立即重排
- ✅ **自动恢复**：鼠标移开后文字恢复原始布局

### 效果示意

```
原始文本：
这是一段很长的文字内容...

图片在中间时：
这是一段很长的  [图片]  文字内容继续
文字...                  在右侧显示
```

## 📦 安装

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择此文件夹

## 🎮 使用

1. 点击浏览器工具栏的插件图标
2. 确保"启用动画"开关已打开
3. 访问任意网页（或打开测试页面）
4. **移动鼠标到段落文字的中间位置**
5. 观察文字如何在图片左右两侧环绕！

## 💡 使用技巧

- **最佳效果**：将鼠标移动到段落的中间位置，而不是边缘
- **观察环绕**：慢慢移动鼠标，看文字如何动态拆分和重组
- **图片大小**：较大的图片（60-100px）环绕效果更明显

## 🎨 自定义

- **上传图片**：点击"选择图片"上传本地图片（推荐透明背景 PNG）
- **调整大小**：拖动"动画大小"滑块（图片越大，环绕效果越明显）
- **调整强度**：控制效果强度
- **调整半径**：控制影响范围

## 🔧 技术实现

本插件使用 pretext 库的核心功能实现文字环绕：

### 1. 文本准备
```javascript
const prepared = pretext.prepareWithSegments(text, font);
```

### 2. 拆分布局
```javascript
// 布局左侧文字
const leftRange = pretext.layoutNextLineRange(prepared, cursor, leftWidth);
const leftLine = pretext.materializeLineRange(prepared, leftRange);

// 布局右侧文字
const rightRange = pretext.layoutNextLineRange(prepared, leftRange.end, rightWidth);
const rightLine = pretext.materializeLineRange(prepared, rightRange);

// 组合：左侧文字 + 空格（图片空间） + 右侧文字
const combinedText = leftLine.text + spaces + rightLine.text;
```

### 3. 实时更新
- 每帧检测鼠标位置
- 计算图片左右两侧的可用宽度
- 使用 pretext 分别布局左右两侧文字
- 用空格填充图片占据的空间
- 更新 DOM 显示新的文本布局

## 🐛 调试

打开浏览器控制台（F12）查看：
- "Text Interaction Animation initialized" - 插件已初始化
- "Scanned X text elements" - 扫描到的文本元素数量
- "Starting animation..." - 动画已启动

## ⚠️ 注意事项

1. **性能**：文字重排是实时的，在文字很多的页面可能会有性能影响
2. **兼容性**：只处理 `<p>`, `<h1-h6>`, `<blockquote>`, `<li>` 等块级文本元素
3. **最佳位置**：将鼠标移动到段落中间位置效果最好

## 🎯 推荐设置

- **图片大小**：60-100px（太小效果不明显，太大影响阅读）
- **最小宽度**：30px（左右两侧的最小文字宽度）
- **测试页面**：使用 `test-wrap.html` 查看最佳效果

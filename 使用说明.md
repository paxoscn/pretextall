# 文字互动动画 Chrome 插件

基于 [pretext](https://github.com/chenglou/pretext) 库实现的 Chrome 浏览器插件。

## 安装

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择此文件夹

## 使用

1. 点击浏览器工具栏的插件图标
2. 确保"启用动画"开关已打开
3. 访问任意网页，移动鼠标
4. 观察动画如何与文字互动！

## 自定义

- **上传图片**：点击"选择图片"上传本地图片
- **调整大小**：拖动"动画大小"滑块
- **调整强度**：拖动"互动强度"滑块
- **调整半径**：拖动"影响半径"滑块

## 技术实现

本插件直接使用 pretext 库的以下功能：

- `prepare()` - 准备文本进行测量
- `layout()` - 计算文本布局
- `prepareWithSegments()` - 准备带分段信息的文本
- `layoutWithLines()` - 获取详细的行布局信息

pretext 提供了高效的文本测量和布局计算，无需频繁的 DOM 操作。

## 文件结构

```
├── manifest.json          # Chrome 扩展配置
├── pretext-loader.js      # Pretext 加载器
├── pretext-lib/           # Pretext 库文件
├── content.js             # 核心动画逻辑
├── content.css            # 动画样式
├── popup.html             # 设置界面
├── popup.js               # 设置逻辑
└── icons/                 # 插件图标
```

# 文字互动动画 Chrome 插件

基于 [pretext](https://github.com/chenglou/pretext) 库实现的 Chrome 浏览器插件，实现**文字避开图片**的动态布局效果。

## ✨ 效果说明

当你移动鼠标到网页文字上时，文字会**实时重新排列**以避开鼠标光标（图片），就像水流绕过石头一样。

## 📦 安装

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择此文件夹

## 🎮 使用

1. 点击浏览器工具栏的插件图标
2. 确保"启用动画"开关已打开
3. 访问任意网页（或打开 `test-simple.html`）
4. **移动鼠标到段落文字上**
5. 观察文字如何重新排列以避开图片！

## 💡 效果说明

- **文字会动态重排**：当鼠标（图片）移动到文字区域时，文字会实时重新布局
- **图片在左侧**：文字从右侧流过
- **图片在右侧**：文字从左侧流过
- **图片移开**：文字恢复原始布局

## 🎨 自定义

- **上传图片**：点击"选择图片"上传本地图片（推荐透明背景 PNG）
- **调整大小**：拖动"动画大小"滑块（图片越大，文字避让越明显）
- **调整强度**：控制效果强度
- **调整半径**：控制影响范围

## 🔧 技术实现

本插件使用 pretext 库的核心功能：

### 1. 文本准备
```javascript
const prepared = pretext.prepareWithSegments(text, font);
```

### 2. 动态布局
```javascript
// 逐行布局，每行根据图片位置计算可用宽度
const lineRange = pretext.layoutNextLineRange(prepared, cursor, availableWidth);
const line = pretext.materializeLineRange(prepared, lineRange);
```

### 3. 实时重排
- 每帧检测鼠标位置
- 计算每行可用宽度（考虑图片遮挡）
- 使用 pretext 重新布局文字
- 更新 DOM 显示新的文本布局

## 🐛 调试

打开浏览器控制台（F12）查看：
- "Text Interaction Animation initialized" - 插件已初始化
- "Scanned X text elements" - 扫描到的文本元素数量
- "Starting animation..." - 动画已启动

## ⚠️ 注意事项

1. **性能**：文字重排是实时的，在文字很多的页面可能会有性能影响
2. **兼容性**：只处理 `<p>`, `<h1-h6>`, `<blockquote>`, `<li>` 等块级文本元素
3. **效果范围**：只有当鼠标移动到文字区域内时才会触发重排

## 🎯 推荐设置

- **图片大小**：60-100px（太小效果不明显，太大影响阅读）
- **测试页面**：使用 `test-simple.html` 查看最佳效果
- **最佳体验**：在段落较长的文章页面上使用

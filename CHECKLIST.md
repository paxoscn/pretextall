## 🐛 问题排查

如果遇到问题，按以下顺序检查：

1. **图标问题**
   - [ ] 检查 `icons` 文件夹
   - [ ] 确认文件名正确
   - [ ] 重新生成图标

2. **安装问题**
   - [ ] 确认 Chrome 版本
   - [ ] 检查开发者模式已开启
   - [ ] 查看扩展管理页面的错误信息

3. **功能问题**
   - [ ] 刷新页面（F5）
   - [ ] 重新加载插件
   - [ ] 检查浏览器控制台（F12）

4. **性能问题**
   - [ ] 减小影响半径
   - [ ] 降低互动强度
   - [ ] 使用更小的图片

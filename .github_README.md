## 📖 简介

这是一个创新的 Chrome 浏览器插件，灵感来源于 [pretext](https://github.com/chenglou/pretext) 项目。它让自定义的动画元素能够跟随鼠标移动，并与网页上的文字产生真实的物理互动效果——当动画靠近文字时，会被文字"推开"，创造出独特而有趣的视觉体验。

### ✨ 核心特性

- 🎨 **自定义动画图片** - 支持上传本地图片作为动画元素
- 🖱️ **智能鼠标跟随** - 动画实时跟随鼠标移动
- 📝 **物理文字互动** - 动画被网页文字"推开"
- ⚙️ **灵活参数配置** - 可调节大小、强度、半径
- 🚀 **性能优化** - 60fps 流畅动画
- 🌐 **全局支持** - 在所有网页上都能使用

## 🎬 效果演示

```
┌─────────────────────────────────┐
│  这是一段文字                    │
│                                 │
│      文字  ←  🎨 (动画被推开)    │
│                                 │
│  更多文字内容                    │
└─────────────────────────────────┘
```

当鼠标移动到文字附近时，动画会被文字产生的"力场"推开，距离越近，排斥力越强。

## 🚀 快速开始

### 安装

1. 下载项目文件
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

### 使用

1. 点击浏览器工具栏的插件图标
2. 确保"启用动画"开关已打开
3. 访问任意网页，移动鼠标
4. 观察动画如何与文字互动！

## 🔧 技术实现

### 核心技术

- **文字检测**: TreeWalker API + Range.getClientRects()
- **物理计算**: 反平方衰减的排斥力模型
- **动画渲染**: requestAnimationFrame + CSS Transform
- **数据存储**: Chrome Storage API
- **消息通信**: Chrome Runtime API

### 性能优化

- ✅ Map 数据结构缓存文字位置
- ✅ GPU 硬件加速（transform）
- ✅ 智能 DOM 变化监听
- ✅ 被动事件监听器
- ✅ 高效的向量计算

## 🐛 常见问题

<details>
<summary><b>动画不显示？</b></summary>

- 检查"启用动画"开关是否打开
- 刷新网页（按 F5）
- 查看浏览器控制台是否有错误（按 F12）
</details>

<details>
<summary><b>效果不明显？</b></summary>

- 增加"互动强度"参数
- 增加"影响半径"参数
- 使用更大、更醒目的图片
</details>

<details>
<summary><b>页面卡顿？</b></summary>

- 减小"影响半径"（降低计算量）
- 降低"互动强度"
- 使用更小的图片文件
</details>

<details>
<summary><b>某些网站不工作？</b></summary>

- Chrome 内部页面（如扩展商店）不支持（安全限制）
- 使用 Shadow DOM 的网站可能部分不支持
- 这是正常现象，尝试其他网站
</details>

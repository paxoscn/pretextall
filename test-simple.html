🎨 文字避开图片效果测试

使用说明

1. 确保插件已启用（点击浏览器工具栏的插件图标）

2. 移动鼠标到下面的段落文字上

3. 观察文字如何重新排列以避开图片（鼠标光标）

4. 尝试慢慢移动鼠标，看文字如何动态调整

测试段落 1

这是一段很长的测试文字，用来演示文字避开图片的效果。当你移动鼠标到这段文字上时， 文字应该会重新排列，在图片周围流动。这个效果使用了 pretext 库的动态布局功能， 可以实时计算每一行的可用宽度，然后重新排列文字。试着慢慢移动鼠标，观察文字如何 优雅地避开图片，就像水流绕过石头一样。这种效果在传统的 CSS 中很难实现， 但是通过 pretext 的强大文本布局能力，我们可以轻松做到。

测试段落 3 - 中文

在这个段落中，我们使用中文来测试文字避开效果。中文的排版和英文有所不同， 每个汉字都是等宽的，这使得排版计算更加简单。当鼠标移动到这段文字上时， 你应该能看到汉字如何重新排列，在图片的左侧或右侧流动。pretext 库支持 多种语言，包括中文、日文、韩文、阿拉伯文等。这个功能对于国际化的网站 非常有用。试着在不同的位置移动鼠标，看看文字如何适应不同的布局。

测试段落 4 - 更长的文本

这是一个更长的段落，包含了更多的文字内容。文字越多，避开效果就越明显。 当图片（鼠标光标）移动到文字区域时，pretext 会计算每一行可以使用的宽度。 如果图片在左侧，文字就会从右侧流过；如果图片在右侧，文字就会从左侧流过。 这个过程是实时的，每一帧都会重新计算。虽然这听起来很消耗性能，但是 pretext 使用了高效的算法，可以在保持流畅的同时提供精确的布局。你可以尝试快速移动 鼠标，看看文字是否能够跟上。在大多数情况下，你应该能看到流畅的动画效果， 文字像液体一样在图片周围流动。这种效果不仅美观，而且非常实用，可以用于 创建更加动态和互动的网页体验。

测试段落 5 - 混合内容

This paragraph contains both English and 中文内容. When you move the mouse over this text, 文字应该会避开图片. The pretext library handles mixed-language content 很好地, ensuring that 每种语言的排版规则 are respected. Try moving the mouse slowly across this paragraph to see how the text reflows around the image. 这个效果在处理多语言内容时特别有用。

调试信息

打开浏览器控制台（F12）查看调试信息。你应该能看到：
- "Text Interaction Animation initialized" - 插件已初始化
- "Scanned X text elements" - 扫描到的文本元素数量
- "Starting animation..." - 动画已启动
